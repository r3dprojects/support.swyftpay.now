# Contributing

Thank you for your interest in swyftpay.now.

## This is an issue-tracker repository

swyftpay.now source code is proprietary. This repository exists solely for public issue tracking — bug reports, feature requests, and questions. We do not accept code pull requests here.

## How to contribute

- **Report a bug** → [Bug Report template](https://github.com/r3dprojects/support.swyftpay.now/issues/new?template=1-bug-report.yml)
- **Request a feature** → [Feature Request template](https://github.com/r3dprojects/support.swyftpay.now/issues/new?template=2-feature-request.yml)
- **Ask a question** → [Question template](https://github.com/r3dprojects/support.swyftpay.now/issues/new?template=3-question.yml)

## What makes a great report

**Bug report:**
- Steps to reproduce (numbered, specific)
- What you expected versus what actually happened
- Your browser, device, and OS version
- Screenshots or recordings if the issue is visual

**Feature request:**
- The problem you are solving, not just the solution you want
- The business or workflow context behind the request
- Alternatives you have already tried or considered

## What happens next

We read every submission. We will close issues with a clear reason if we decide not to act on them — we aim to be transparent about prioritisation. We cannot commit to specific timelines.

## Conduct

We follow the [Contributor Covenant 2.1](CODE_OF_CONDUCT.md). Respectful, constructive communication is expected of everyone in this space.
